A row in the sidebar case list. HealthPath calls past conversations "cases" because each one is a real bureaucratic situation.

```jsx
<ConversationItem
  title="Medication denied by EPS"
  meta="2 days ago · 4 messages"
  status="waiting"
  active
/>
```

Status: `open` (teal — active back-and-forth), `waiting` (amber — waiting on an office). No dot = closed/resolved.
